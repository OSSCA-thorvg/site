# #3788 text: support ttc extension

- Link: https://github.com/thorvg/thorvg/issues/3788
- 난이도: 63/100
- 실현 가능성: 중간
- 초심자 추천: 조건부 추천
- 관련 영역: SFNT loader, TTC header/table offset, face selection, binary validation
- 배울 수 있는 것: big-endian parser, offset base, overflow 검증, font collection

## 이슈 요약

`.ttc`/`.otc`는 loader로 도달하지만 `ttcf` container를 해석하지 못해 실패한다. 실패 지점은 좁지만 TTC table offset 기준과 여러 face 중 선택 규칙을 정해야 하므로 단순 signature 추가만으로 끝나지 않는다.

## main 코드 조사

**확인된 증거**

- `LoaderMgr::_findByPath()`는 `ttc`와 `otc`를 이미 `FileType::Sfnt`로 분류한다.
- `SfntLoader::gen()`은 `0x00010000`, `true`, `OTTO`만 허용하고 `ttcf`를 거절한다.
- `SfntReader::table()`은 offset table이 `data+0`에 있다고 보고 directory를 `data+12`에서 검색한다.
- TTC의 face directory 위치와 table record의 원본-file 기준 offset을 따로 표현할 field가 현재 reader에 없다.

```cpp
// src/loaders/sfnt/tvgSfntLoader.cpp
if (type == 0x00010000 || type == 0x74727565) return new TtfReader(data, size);
else if (type == 0x4F54544F) return new OtfReader(data, size);
// 0x74746366 ('ttcf') case가 없다.
```

```text
TTC base
  ├─ ttcf header + faceOffsets[]
  └─ faceOffsets[i] -> SFNT directory
                       └─ table record offset는 원본 TTC base 기준
```

## 원인 가설과 확인 방법

- **확정:** extension dispatch는 정상이고 `ttcf` container 처리와 reader face base가 없다.
- **가설:** 보고 파일은 첫 face만 읽어도 표시되지만 원하는 family가 다른 face일 수 있다.
- **확인 방법:** face별 family/name table이 다른 작은 TTC를 사용해 first-face MVP와 명시적 face 선택 요구를 분리한다.

## 수정 방향 계획

1. 재배포 가능한 TTC v1 fixture와 malformed count/offset fixture를 만든다.
2. reader에 원본 data base와 face-directory offset을 분리하고 table record offset은 원본 base에서 검증한다.
3. 선택 face의 scaler signature로 `TtfReader`/`OtfReader`를 고른다.
4. 첫 face MVP를 승인받거나, index/name 선택이 필요하면 C++/CAPI API 설계를 별도 이슈로 분리한다.

## 실현 가능성 판단

첫 face 지원은 **중간**이며 binary parser를 배우는 초심자에게 조건부로 적합하다. 여러 face public API까지 포함하면 ABI·font selection 정책 때문에 난도가 크게 오른다.

## 위험/검증

- `numFonts * 4`, face offset, directory/table offset의 overflow와 범위를 검사한다.
- TTC v1/v2, 내부 TTF/OTF, 기존 단일 TTF/OTF를 함께 테스트한다.
- reader가 shifted pointer를 소유해 원본 allocation 해제 주소를 잃지 않도록 한다.
